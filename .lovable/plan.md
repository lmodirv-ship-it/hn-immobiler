

# جعل الموقع عالمياً 🌍

الهدف: تحويل HN Immobilier من منصة محلية مغربية إلى منصة عقارية عالمية متاحة بعدة لغات وعملات وطرق دفع دولية.

## المحاور الرئيسية

### 1. اللغات المتعددة (Multi-language)
حالياً: فرنسي + عربي فقط.
الإضافة:
- **English** (إجباري للوصول العالمي)
- **Español** (إسبانيا قريبة من المغرب + أمريكا اللاتينية)
- **Deutsch** (مستثمرون ألمان في العقارات)
- توسيع `src/lib/i18n.ts` ليشمل 5 لغات
- تحديث `LanguageContext` و `Header` لعرض كل اللغات في القائمة المنسدلة
- دعم RTL تلقائي للعربية

### 2. العملات المتعددة (Multi-currency)
- إنشاء `useCurrency` hook مع تحويل تلقائي حسب الدولة
- العملات المدعومة: **MAD, EUR, USD, GBP, AED, SAR**
- استخدام API مجاني لأسعار الصرف (exchangerate-api أو frankfurter.app)
- مبدّل عملة في الهيدر بجانب اللغة
- عرض السعر بعملة الزائر تلقائياً مع إمكانية التغيير

### 3. الكشف الجغرافي الذكي (موجود جزئياً)
- توسيع `useGeoLocation` ليقترح **اللغة + العملة + طريقة الدفع** المناسبة
- خريطة دول → لغة افتراضية موسعة (EN للدول الأنغلوسكسونية، ES للإسبانية، إلخ)

### 4. الـ SEO الدولي
- تحديث `SEO.tsx` لدعم **hreflang tags** لكل لغة
- إضافة `<link rel="alternate" hreflang="x">` لكل صفحة
- توسيع `sitemap` edge function لإنتاج روابط متعددة اللغات
- تحديث `robots.txt` و meta tags بـ Open Graph متعدد اللغات

### 5. طرق الدفع الدولية (موجود جزئياً)
- Stripe و PayPal جاهزان (دفع دولي ✓)
- إضافة عرض الأسعار بعملة الزائر في `Pricing.tsx` و `Checkout.tsx`
- إخفاء طرق الدفع المحلية (CMI, PayZone, Wafacash) للزوار خارج المغرب — أو وضعها في تبويب "Morocco Only"

### 6. تحسينات إضافية
- **أرقام الهاتف الدولية**: استخدام `react-phone-input-2` في النماذج
- **المنطقة الزمنية**: عرض أوقات المعاينات بتوقيت الزائر المحلي
- **عرض المسافات**: km مقابل miles حسب الدولة
- **تحديث Footer** ليشمل روابط متعددة اللغات وعنوان دولي

## الملفات المتأثرة

**سيتم تعديلها:**
- `src/lib/i18n.ts` — إضافة EN/ES/DE
- `src/contexts/LanguageContext.tsx` — دعم 5 لغات
- `src/components/Header.tsx` — قائمة لغات + مبدّل عملة
- `src/components/SEO.tsx` — hreflang tags
- `src/hooks/useGeoLocation.ts` — اقتراح عملة + توسيع خريطة الدول
- `src/pages/Pricing.tsx` و `src/pages/Checkout.tsx` — عرض متعدد العملات
- `supabase/functions/sitemap/index.ts` — sitemap متعدد اللغات
- `index.html` — meta tags دولية

**سيتم إنشاؤها:**
- `src/contexts/CurrencyContext.tsx` — مزود العملة
- `src/hooks/useCurrency.ts` — تحويل وتنسيق الأسعار
- `src/components/CurrencySelector.tsx` — مبدّل العملة في الهيدر
- `src/lib/currencies.ts` — قائمة العملات والرموز

## التنفيذ المرحلي (مقترح)

نظراً لحجم العمل، الأفضل تنفيذها على **3 مراحل**:

| المرحلة | المحتوى | الأولوية |
|---|---|---|
| **1** | اللغات الـ 5 + كشف جغرافي موسع + SEO hreflang | 🔥 عالية |
| **2** | نظام العملات المتعددة + تحويل تلقائي | ⭐ متوسطة |
| **3** | تحسينات (هاتف دولي، منطقة زمنية، Footer) | ✨ منخفضة |

## أسئلة قبل التنفيذ

1. **هل تريد البدء بالمراحل الثلاث كلها أم المرحلة 1 فقط؟** (المرحلة 1 وحدها تعطي تأثيراً قوياً وسريعاً)
2. **اللغات المطلوبة:** هل تكفي EN/ES/DE أم تريد إضافة لغات أخرى (PT, IT, ZH, RU)؟
3. **العملة الافتراضية للعرض:** MAD دائماً مع تحويل، أم عملة الزائر مباشرة؟
